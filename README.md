# 新年祝福贺卡使用文档

## 项目简介

这是一个精美的新年祝福贺卡网页，包含了多种动态效果和交互功能，适合在新年期间分享给朋友和家人，传递新年的祝福和温暖。

## 网页分块结构

### 1. 开屏动画
- **位置**：页面加载时显示
- **内容**：樱花风格的加载动画，包含进度条和加载提示
- **功能**：为用户提供视觉反馈，增强页面加载体验

### 2. 背景容器
- **位置**：整个页面的背景
- **内容**：包含三种背景模式（白天、黄昏、黑夜）
- **功能**：通过切换按钮可以在不同背景模式间切换，营造不同的氛围

### 3. 贺卡内容
- **位置**：页面中央上方
- **内容**：
  - 标题："新年快乐！"
  - 副标题："愿新的一年里，万事如意，幸福安康！"
  - 背景切换按钮：简约风格的日/夜模式切换开关
- **功能**：展示贺卡的核心祝福信息，提供背景模式切换功能

### 4. 音乐播放器
- **位置**：贺卡内容下方
- **内容**：
  - 当前播放歌曲信息
  - 播放控制按钮（上一首、播放/暂停、下一首）
  - 进度条和时间显示
  - 可排序和拖拽的歌曲列表
- **功能**：播放背景音乐，支持歌曲切换、进度调整、列表排序和拖拽重排

### 5. 图片展示区域
- **位置**：音乐播放器下方
- **内容**：
  - 标题："留影集"
  - 淡入淡出轮播区域：展示贺卡素材图片
  - 轮播指示器：显示当前图片位置
  - 左右滑动图片列表：展示所有贺卡素材图片
- **功能**：以轮播和列表形式展示图片，支持手动切换和自动轮播

### 6. 页脚祝福语
- **位置**：页面底部
- **内容**："㊗️ 新年快乐，美满幸福！"（可点击跳转到B站个人主页）
- **功能**：再次强调新年祝福，提供B站个人主页链接

## 实现的功能

### 1. 视觉效果
- **动态背景**：三种背景模式（白天、黄昏、黑夜），白天有雪花效果，黑夜有烟花效果
- **樱花流星雨**：页面顶部飘落的樱花效果
- **烟花动画**：黑夜模式下的烟花绽放效果
- **淡入淡出轮播**：图片的平滑切换效果

### 2. 交互功能
- **背景模式切换**：点击简约风格的开关按钮在日/夜模式间切换
- **音乐控制**：播放/暂停、上一首、下一首、进度调整
- **歌曲管理**：歌曲列表排序（升序/降序）、拖拽重排、重置顺序
- **图片浏览**：手动切换图片、自动轮播、点击列表项查看图片
- **超链接跳转**：点击页脚祝福语跳转到B站个人主页

### 3. 技术特性
- **响应式设计**：适配不同屏幕尺寸
- **平滑动画**：各种过渡效果和动画
- **用户友好**：直观的操作界面和清晰的视觉反馈
- **代码组织**：模块化的Vue 3代码结构，易于维护和扩展

## 使用方法

### 本地使用
1. 下载或克隆整个项目到本地
2. 确保项目文件结构完整，包含以下文件和文件夹：
   - `index.html`：主页面文件
   - `app.js`：Vue 3应用脚本
   - `style.css`：样式文件
   - `贺卡素材/`：存放图片素材的文件夹
   - `音乐素材/`：存放音乐文件的文件夹
3. 双击 `index.html` 文件，在浏览器中打开即可

### 部署到GitHub Pages
1. 在GitHub上创建一个新的仓库
2. 将项目文件上传到仓库中
3. 在仓库的 "Settings" 页面中，找到 "Pages" 选项
4. 在 "Source" 下拉菜单中选择 "main" 分支，然后点击 "Save"
5. 等待几分钟后，GitHub Pages会自动构建和部署网站
6. 在 "Pages" 页面中找到生成的网站URL，即可访问部署后的贺卡

## 功能使用指南

### 1. 背景模式切换
- **操作**：点击贺卡内容区域的圆形切换按钮
- **效果**：页面背景会在白天（雪花效果）和黑夜（烟花效果）之间切换

### 2. 音乐播放器使用
- **播放/暂停**：点击中间的大按钮
- **切换歌曲**：点击左右两侧的 "上一首" 和 "下一首" 按钮
- **调整进度**：拖动进度条到指定位置
- **排序歌曲**：点击歌曲列表上方的 "名称↑"/"名称↓" 按钮
- **重置顺序**：点击歌曲列表上方的 "重置顺序" 按钮
- **拖拽重排**：点击并拖动歌曲项到新的位置

### 3. 图片浏览
- **自动轮播**：页面加载后会自动开始轮播图片
- **手动切换**：点击轮播区域左右两侧的箭头按钮
- **点击指示器**：点击轮播下方的小圆点快速跳转到对应图片
- **查看列表**：在下方的图片列表中点击任意图片查看

### 4. 访问B站主页
- **操作**：点击页面底部的 "㊗️ 新年快乐，美满幸福！" 文字
- **效果**：会在新标签页中打开B站个人主页

## 注意事项

1. **浏览器兼容性**：建议使用Chrome、Firefox、Edge等现代浏览器访问，以获得最佳体验

2. **文件路径**：确保项目文件结构完整，特别是 `贺卡素材/` 和 `音乐素材/` 文件夹中的文件必须存在

3. **GitHub Pages部署**：
   - 部署后可能需要等待几分钟才能正常访问
   - 确保所有文件都已正确上传到GitHub
   - 检查仓库设置中的Pages配置是否正确

4. **音频播放**：
   - 由于浏览器的自动播放策略，首次播放音乐可能需要用户交互
   - 部分浏览器可能会限制音频自动播放

5. **图片加载**：
   - 首次加载时，图片可能需要一些时间才能完全加载
   - 确保图片文件格式正确且大小适中，以提高加载速度

## 自定义指南

### 1. 修改贺卡内容
- 编辑 `index.html` 文件中的标题和副标题文本

### 2. 添加/修改音乐
- 在 `音乐素材/` 文件夹中添加音乐文件
- 编辑 `app.js` 文件中的 `songs` 数组，添加对应的歌曲信息

### 3. 添加/修改图片
- 在 `贺卡素材/` 文件夹中添加图片文件
- 编辑 `app.js` 文件中的 `images` 数组，添加对应的图片信息

### 4. 修改B站链接
- 编辑 `index.html` 文件中页脚的 `<a>` 标签，将 `href` 属性值改为你的B站个人主页链接

### 5. 调整样式
- 编辑 `style.css` 文件，修改对应的样式规则以调整页面外观
